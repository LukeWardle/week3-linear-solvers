 # Week 3: Linear System Solvers 
 
Production-quality linear system solver demonstrating: 
- Multiple solving strategies (direct, LU decomposition, least-squares) 
- Numerical stability checking (condition number) 
- Professional code structure, type hints, and docstrings 
- Comprehensive test suite 
 
Built as part of the AI Engineering Foundations Programme, Week 3. 
 
## Installation 
 
``` 
python -m venv venv 
venv\Scripts\activate  (Windows) 
pip install -r requirements.txt 
``` 
 
## Running Tests 
 
``` 
pytest test_solvers.py -v 
```
